# Stress Prediction Model

A machine learning project to predict **Total Perceived Stress (TPStress)** using psychological and demographic features.

## Overview

This project explores how psychological constructs — such as optimism, mastery, affect, life satisfaction, self-esteem, and intolerance of uncertainty — can predict perceived stress levels. It covers the full ML workflow from data preparation to model evaluation.

## Features

- age
- Total Optimism
- Total Mastery
- Total Positive Affect
- Total Negative Affect
- Life Satisfaction (tlifesat)
- Total Self Esteem
- Total Social Desirability
- Total PCOISS

## Objectives

- Explore the dataset (distributions, outliers, missing values)
- Perform descriptive statistics and correlation analysis
- Standardize/normalize variables
- Build predictive models using different regression approaches
- Evaluate performance with MSE and R²
- Compare models and select the best one
- Interpret results

## Technologies

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
